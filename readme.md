# 📊 Bookwise - 智能記帳管理系統

一個現代化的全端記帳應用程式，提供直觀的用戶界面和強大的財務管理功能。

## 🌐 線上體驗


## 🚀 快速開始

### 1. 訪問應用
直接前往 **https://bookwise-two-omega.vercel.app** 開始使用

### 2. 註冊新帳戶
- 點擊「立即註冊」
- 填寫使用者名稱、電子信箱和密碼
- 完成註冊後自動登入

### 3. 或使用測試帳戶
如果已有測試資料，可直接登入體驗

## ✨ 主要功能

### 💰 記帳管理
- **新增記帳記錄**：支援收入/支出分類記錄
- **智能分類**：10+ 預設分類，包含餐飲、交通、購物等
- **快速搜尋**：按日期、分類、金額範圍篩選
- **編輯/刪除**：隨時修改或移除記錄

### 📈 統計分析
- **收支概覽**：當月收入、支出、結餘統計
- **分類分析**：各分類使用頻率和金額分布
- **趨勢圖表**：視覺化財務趨勢
- **時間篩選**：自定義時間範圍分析

### 🏷️ 分類管理
- **自訂分類**：新增個人化收支分類
- **顏色標識**：為每個分類設定專屬顏色
- **使用統計**：查看各分類的使用次數
- **編輯管理**：修改分類名稱和屬性

### 👤 用戶管理
- **帳戶設定**：修改個人資料
- **密碼管理**：安全的密碼更新
- **登入記錄**：查看帳戶活動

## 🎯 使用指南

### 🏠 儀表板 (Dashboard)
1. **登入後的首頁**，顯示財務概覽
2. **快速新增**：點擊「+」按鈕快速記帳
3. **近期記錄**：查看最新的 5 筆交易
4. **統計卡片**：一目了然的收支狀況

### 📝 記帳記錄 (Expenses)
1. **查看所有記錄**：完整的交易清單
2. **新增記錄**：
   - 選擇收入/支出類型
   - 選擇分類
   - 輸入金額和描述
   - 設定交易日期
3. **搜尋篩選**：
   - 按分類篩選
   - 設定日期範圍
   - 選擇收入/支出類型
4. **批量操作**：選擇多筆記錄進行刪除

### 📊 統計分析 (Statistics)
1. **總覽統計**：當期收入、支出、結餘
2. **分類統計**：
   - 各分類金額排行
   - 使用頻率分析
   - 視覺化圓餅圖
3. **時間篩選**：
   - 選擇月份或自定義範圍
   - 比較不同時期數據

### 🏷️ 分類管理 (Categories)
1. **查看所有分類**：收入和支出分類列表
2. **新增分類**：
   - 設定分類名稱
   - 選擇收入或支出類型
   - 選擇代表顏色
3. **編輯分類**：修改現有分類資訊
4. **刪除分類**：移除不需要的分類

### ⚙️ 個人設定 (Profile)
1. **基本資料**：修改使用者名稱和信箱
2. **密碼變更**：安全的密碼更新流程
3. **帳戶資訊**：查看註冊時間等資訊

## 🛠️ 技術架構

### 前端技術
- **React 18** - 現代化前端框架
- **Vite** - 快速建置工具
- **Material-UI (MUI)** - 現代化 UI 組件庫
- **React Router** - 客戶端路由管理
- **Axios** - HTTP 請求處理

### 後端技術
- **Node.js** - JavaScript 運行環境
- **Express.js** - Web 應用框架
- **MySQL** - 關聯式資料庫
- **JWT** - 身份驗證
- **bcrypt** - 密碼加密
- **Winston** - 日誌系統

### 部署平台
- **前端**：Vercel - 全球 CDN 部署
- **後端**：Railway - 容器化部署
- **資料庫**：Railway MySQL - 雲端資料庫

## 🆘 常見問題

### Q: 忘記密碼怎麼辦？
A: 目前請聯繫管理員重設密碼，未來版本將提供自助重設功能。

### Q: 可以匯出數據嗎？
A: 數據匯出功能正在開發中，將支援 CSV/Excel 格式。

### Q: 支援多幣別嗎？
A: 目前以新台幣 (TWD) 為主，多幣別功能在規劃中。

### Q: 數據會被永久保存嗎？
A: 是的，所有記帳數據都會安全保存在雲端資料庫中。
